# Shraddha Wakchaure — Portfolio

A premium, production-ready portfolio website built with modern web technologies.

## Tech Stack

- **React 18** — UI framework
- **Vite 5** — Build tool & dev server
- **Tailwind CSS v3** — Utility-first styling
- **Framer Motion 11** — Animations
- **lucide-react** — Icons

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Features

- Typewriter effect cycling through roles
- IntersectionObserver-powered nav highlighting
- Animated stat counters on scroll
- Magnetic button hover effects
- Staggered grid entrance animations
- Alternating timeline layout
- Glassmorphism card design
- Full mobile responsive design
- Contact form with success state
- Dark, precise design aesthetic

## Deploy

Configured for Vercel SPA deployment with `vercel.json` rewrite rules.

```bash
npx vercel
```
